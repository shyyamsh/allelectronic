This is a small, self-contained Python example intended for beginners. It contains:

- A simple `converters` module with functions for common unit conversions.
- A CLI (`main.py`) that demonstrates using the converters interactively.
- Unit tests under `tests/` using the standard `unittest` framework.

Quick start

1. Run the CLI interactively:

```bash
python3 python_example/main.py
```

2. Run tests:

```bash
python3 -m unittest discover -s python_example -p "test_*.py" -v
```

This project is dependency-free and works with Python 3.10+.
